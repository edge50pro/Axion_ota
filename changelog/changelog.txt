# Device changelog:
Device:
• Moto G84 - bangkk
Status:
• Stable
Build date:
• 21/05/26
Changed:
• Partitions ext4 for erofs, For greater speed and agility of the system for extracting files, compressing, etc.
• GApps type build
• Modification of lineage_interfaces resources, removing everything except the touch/aidl directory and Android.bp to resolve conflicts with duplicates from the source.
• Adding domain of sepolicy for hal_lineage_powershare_default.te and hal_lineage_touch_default.te in hardware/motorola
Add:
• OTA updates are now possible; you can update directly from the settings using Dirty Flash.
• Motocamera & DolbySound - Sony
• The Nexus Pixel launcher has been added instead of the Clover launcher.
• Added an overlay for the updater to check for updates in the Clover repository for bangkk on my Github.
• OTAs will be downloaded via ServerHive, resulting in faster download speeds and fewer delays or errors.
